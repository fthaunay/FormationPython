DataScience Challenge 7

Classification automatique des modèles de voitures

Lors du DataScience Challenge 1, nous avons produit un jeu de données de 64 553 modèles de voitures et leur 190 caractéristiques techniques. Dans ce challenge je vous propose de classifier les voitures en fonction de leur catégorie que vous trouverez dans les champs (bus, cabriolet, coupe, hatchback, mvp, pickup, sedan, stationwagon, suv_crossover).

Les données se trouvent dans le fichier carsdata_features.csv. Le séparateur de champs est ','.

  1. Quelle est la catégorie de voiture la plus fréquente ?
  2. Quelles caractéristiques techniques peut-on utiliser pour faire de la classification ?
  3. Quelle est l'éfficacité de classification que vous pouvez atteindre ?
